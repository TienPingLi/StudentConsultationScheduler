# 學生諮詢預約系統 Student Consultation Appointment System

本專案是 C# Windows Forms 期末專題，使用 SQL Server Express LocalDB 作為資料庫。
新版已將「可視化課表」改成登入後的主畫面，使用者可以直接在課表上看老師可諮詢時間、已預約時段、預約狀態，並點擊格子新增或查看預約。

## 開發環境

- Visual Studio 2022
- .NET Framework 4.8
- C# Windows Forms
- SQL Server Express LocalDB

## 預設登入帳號

| 帳號 | 密碼 |

## 主要功能

1. 管理員登入。
2. 登入紀錄寫入 `Logs/login_log.txt`。
3. 學生資料新增、修改、刪除。
4. 學號不可重複，避免建立兩筆相同學生。
5. 老師資料新增、修改、刪除。
6. 老師可諮詢時間使用視覺化課表設定，點擊格子即可開放或取消。
7. 諮詢時間只能選擇整點時段，例如 `09:00-10:00`。
8. 登入後主畫面就是可視化課表，不需要另外打開課表視窗。
9. 主畫面可依預約週、科目、老師、是否有預約、預約狀態篩選。
10. 每位老師使用不同顏色顯示，同一格有多位老師時會用分隔線切開。
11. 點擊「可預約」時段可直接新增預約，並自動帶入老師、日期與時段。
12. 點擊已預約時段會跳出「預約詳細資訊」彈窗。
13. 預約詳細資訊彈窗會顯示學生、學號、班級、老師、科目、辦公室、日期、時間、主題、備註與狀態。
14. 預約狀態管理：已預約、已完成、已取消。
15. 可以在詳細資訊彈窗或主畫面右側列表快速標記完成或取消預約。
16. 同一位老師同一天同時段不可重複預約。
17. 同一位學生同一天同時段不可預約兩位老師。
18. 老師沒有開放的星期與時段不能新增預約。
19. 右側改為精簡的本週預約列表，可雙擊查看詳細資料。
20. 關鍵字搜尋：學生、學號、老師、科目、主題、狀態。
21. 登入後自動檢查 15 分鐘內即將開始的預約。
22. SQL Server LocalDB 自動建立資料庫與測試資料。
23. 小遊戲中心：整合五張撲克、西洋棋小遊戲與簡易電子琴。
24. 背景音樂可切換：將其他 `.wav` 音樂檔放入 `Assets` 資料夾，重新建置後即可在主畫面「曲目」下拉選單切換。

## 執行方式

1. 使用 Visual Studio 2022 開啟 `StudentConsultationScheduler.sln`。
2. 確認電腦已安裝 SQL Server Express LocalDB。
3. 按下 `F5` 執行。
4. 第一次執行時，程式會在使用者 LocalAppData 資料夾自動建立 LocalDB 資料庫，避免重新解壓縮或搬移專案後找不到 MDF 檔。
5. 使用預設帳號 `admin`、密碼 `1234` 登入。

## 使用流程

1. 登入系統後會直接進入「可視化課表主畫面」。
2. 上方可以選擇預約週、科目、老師、是否有預約、預約狀態。
3. 課表中顯示老師可諮詢時段，每位老師使用不同顏色。
4. 點擊「可預約」格子會開啟新增預約視窗，並自動帶入老師、日期與整點時段。
5. 點擊已預約格子會開啟預約詳細資訊彈窗。
6. 在詳細資訊彈窗可以修改預約、標記已完成、取消預約或恢復為已預約。
7. 右側本週預約列表可以雙擊查看詳細資訊，也可以用上方按鈕標記完成或取消。
8. 老師管理中可以用「用課表設定可諮詢時間」設定老師可開放的星期與整點時段。

## DEMO 影片建議流程

1. 登入系統，展示主畫面已經是可視化課表。
2. 展示上方篩選：預約週、科目、老師、是否有預約、預約狀態。
3. 進入學生管理，新增學生並示範學號不可重複。
4. 進入老師管理，示範用視覺化課表設定老師可諮詢時間。
5. 回到主畫面，點擊可預約時段，快速建立預約。
6. 點擊已預約時段，展示預約詳細資訊彈窗。
7. 在彈窗中把預約標記為已完成，再用狀態篩選查看。
8. 示範取消預約後，該時段可重新預約。
9. 示範同一學生同時段預約另一位老師會被阻擋。
10. 展示右側精簡本週預約列表，說明雙擊可查看預約詳細資料。

## LocalDB 錯誤修正說明

新版資料庫名稱為 `StudentConsultationSchedulerDB_Stable`，並將 MDF/LDF 放到 `%LOCALAPPDATA%\StudentConsultationScheduler\App_Data`，避免舊版資料庫路徑失效後造成：

```text
Cannot open database "StudentConsultationDB" requested by the login.
```

若你之前執行過舊版，請先刪除舊資料夾，再重新解壓縮本 ZIP，並用 Visual Studio 開啟新版 `.sln` 後執行。

## 本版新增：不可預約過去時間
- 新增或修改預約時，若狀態為「已預約」，日期與開始時間必須晚於目前時間。
- 可視化課表中已經過去的可諮詢時段會顯示為「已過期」，點擊時不會開啟新增預約。
- 已取消或已完成的歷史資料仍可保留，用於查詢與紀錄。

## 本版新增：背景音樂可切換

- 主畫面上方新增「曲目」下拉選單。
- 程式會自動讀取輸出資料夾 `Assets` 內的所有 `.wav` 檔。
- 選擇不同曲目時，如果背景音樂正在播放，會立即切換到新曲目並循環播放。
- 若音樂目前是關閉狀態，切換曲目只會記住選擇，不會自動播放。
- 你可以把新的 `.wav` 檔放到專案的 `StudentConsultationScheduler/Assets/` 資料夾；本專案已設定 `Assets\*.wav` 會在建置時自動複製到輸出資料夾。
- 注意：此功能使用 `System.Media.SoundPlayer`，只支援 `.wav`，不支援 `.mp3`。

## 本版新增：UI 美化與人性化操作

- 全系統主要表單套用一致的字體、按鈕、表格與輸入框樣式。

- 全部預約 / 搜尋頁面加入操作提示，輸入關鍵字後可直接按 Enter 搜尋。


## 2026-06-17 LocalDB 路徑修正版

本版修正資料庫建立位置，不再將 `.mdf` / `.ldf` 建在 `bin\Debug\App_Data`。資料庫會建立在：

```text
%LOCALAPPDATA%\StudentConsultationScheduler\App_Data
```

這樣即使專案資料夾重新解壓縮、改名或搬移，也不會再因為舊的 `bin\Debug` 路徑不存在而導致 `Unable to open the physical file`。另外音樂檔複製方式已改為 `PreserveNewest`，建置速度會比較快。
